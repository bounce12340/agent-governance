# Security Policy / 安全性政策

## Reporting a vulnerability / 回報弱點

If you find a security issue, please report it responsibly.

如果你發現安全問題，請負責任地回報。

Please include:

- a short description
- affected files or endpoints
- steps to reproduce
- expected impact
- any proof of concept if safe to share

請包含：

- 簡短描述
- 影響的檔案或端點
- 重現步驟
- 預期影響
- 若安全可分享的 PoC

## What not to share publicly / 不要公開分享

- secrets or tokens
- private user data
- live exploit details before disclosure is complete

- secrets 或 token
- 私人用戶資料
- 尚未完成揭露前的利用細節

## Response expectations / 回應方式

- We will acknowledge the report.
- We will review the issue.
- We may ask for clarification or extra evidence.

- 我們會確認收到。
- 我們會檢查問題。
- 必要時會請你補充資訊或證據。
