# Code of Conduct / 行為準則

## Our Pledge / 我們的承諾

We want this project to be welcoming, respectful, and useful.

我們希望這個專案是友善、尊重、而且真的有幫助的。

## Standards / 標準

### English
- Be respectful
- Stay constructive
- Give credit when due
- Keep feedback focused on the work
- Avoid harassment, hate, or personal attacks

### 繁體中文
- 保持尊重
- 保持建設性
- 該給 credit 的就給
- 回饋請聚焦在內容本身
- 不要騷擾、仇恨、或做人身攻擊

## Enforcement / 執行

### English
Project maintainers may remove or reject contributions that violate this code of conduct.

### 繁體中文
維護者可以移除或拒絕違反本準則的內容。
