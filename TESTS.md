# Minimal test matrix
# 最小測試矩陣

## English

1. README bilingual coverage
   - Pass if both English and Traditional Chinese sections exist.

2. About page governance summary
   - Pass if separation of powers and harness engineering are explained.

3. Example flow completeness
   - Pass if legislative, executive, and judiciary are all represented.

## 繁體中文

1. README 雙語覆蓋
   - 只要同時有英文與繁中區塊就算通過。

2. About 的制度摘要
   - 只要有解釋三權分立與 Harness Engineering 就算通過。

3. 範例流程完整性
   - 只要有立法權、行政權、司法權三個角色就算通過。
